# Part 1 — String Foundations

# 1. How Strings Work in Java (Internally)

## The Most Important Thing to Know

```
In Java, String is IMMUTABLE.
Every time you modify a string, Java creates a NEW string object.

String s = "hello";
s = s + " world";   // Did NOT modify "hello"
                     // Created brand new "hello world" in memory
                     // Old "hello" is now garbage

This means: String concatenation in a loop is O(n²)!

String result = "";
for (int i = 0; i < n; i++) {
    result += str.charAt(i);   // n concatenations, each O(n) → O(n²) total!
}

ALWAYS use StringBuilder for building strings:
StringBuilder sb = new StringBuilder();
for (int i = 0; i < n; i++) {
    sb.append(str.charAt(i));  // O(1) per append → O(n) total!
}
String result = sb.toString();
```

## String Pool and == vs .equals()

```java
String a = "hello";
String b = "hello";
String c = new String("hello");

a == b;        // TRUE  — same reference in String Pool
a == c;        // FALSE — c is a new object on heap
a.equals(c);   // TRUE  — same content

// RULE: ALWAYS use .equals() to compare string content
// NEVER use == unless you specifically want reference equality

// Safe comparison (handles null):
Objects.equals(a, b);   // true if both null, or both same content
```

## Character Arithmetic — The Most Useful Trick

```java
// Characters are just integers in Java!
char c = 'A';
int val = c;        // val = 65 (ASCII of 'A')

// Convert char to int index (0-25 for a-z):
int idx = 'e' - 'a';   // = 4  (e is the 5th letter, 0-indexed)
int idx2 = 'Z' - 'A';  // = 25

// Convert digit char to int:
char digit = '7';
int num = digit - '0';  // = 7

// Convert int to char:
char letter = (char)('a' + 4);  // = 'e'
char digitChar = (char)('0' + 7); // = '7'

// Check character type:
Character.isDigit('5');      // true
Character.isLetter('a');     // true
Character.isLetterOrDigit('_'); // false
Character.isWhitespace(' '); // true
Character.isUpperCase('A');  // true
Character.toLowerCase('A');  // 'a'
Character.toUpperCase('a');  // 'A'
```

# 2. String vs StringBuilder vs char[] — When to Use What

```
STRING:
→ Use when: read-only, comparisons, passing as parameter
→ Immutable, thread-safe
→ DON'T use for building/modifying

STRINGBUILDER:
→ Use when: building a result string, reversing, appending in loop
→ Mutable, O(1) append, O(n) toString
→ GO-TO choice for string construction in interviews

CHAR ARRAY:
→ Use when: in-place modification needed, two-pointer on chars
→ char[] arr = s.toCharArray();  → O(n) conversion
→ Fastest for index-based access and swap
→ Convert back: new String(arr)

STRING ARRAY:
→ Use when: splitting into words, tokenizing
→ String[] words = s.split(" ");

MEMORY COMPARISON:
String s = "hello";              // Immutable, in String Pool
StringBuilder sb = new StringBuilder("hello"); // Mutable, on heap
char[] arr = s.toCharArray();    // Separate mutable array on heap
```

## StringBuilder Complete API

```java
StringBuilder sb = new StringBuilder();

// BUILD
sb.append("hello");       // Append string
sb.append(' ');           // Append char
sb.append(42);            // Append int (converts to "42")
```

# 4. Common String Pitfalls and How to Avoid Them

## Pitfall 1: String Comparison with ==

```java
// ❌ WRONG
if (s1 == s2) { ... }  // Compares REFERENCES, not content

// ✅ CORRECT
if (s1.equals(s2)) { ... }
if (s1.equalsIgnoreCase(s2)) { ... }  // Case-insensitive
```

## Pitfall 2: Slow String Concatenation in Loop

```java
// ❌ WRONG — O(n²)
String result = "";
for (char c : chars) {
    result += c;  // Creates new string each time!
}

// ✅ CORRECT — O(n)
StringBuilder sb = new StringBuilder();
for (char c : chars) {
    sb.append(c);
}
String result = sb.toString();
```

## Pitfall 3: Off-by-One in substring()

```java
String s = "hello";
//          01234

s.substring(1, 3);  // "el"  — [1, 3) exclusive end!
s.substring(1, 4);  // "ell" — NOT "ello"
s.substring(1);     // "ello" — from 1 to end

// REMEMBER: substring(start, end) means chars at index start, start+1, ..., end-1
// Length of result = end - start
```

## Pitfall 4: Not Handling Edge Cases

```java
// Always check for:
if (s == null || s.isEmpty()) return "";  // Null and empty
if (s.length() == 1) return s;            // Single character
// Palindrome check for even/odd length
// Two-pointer edge: left < right (not <=)
```

## Pitfall 5: Using String When char[] Needed for Swap

```java
// Strings are immutable — can't swap characters
String s = "hello";
// s[0] = 'H';  // ❌ WON'T COMPILE

// Use char array for in-place modification:
char[] arr = s.toCharArray();
char temp = arr[0];
arr[0] = arr[4];
arr[4] = temp;
String result = new String(arr);  // "oellh"
```

# 5. Pattern 1 — Two Pointers on Strings

## When to Use

```
Keywords: palindrome, reverse, symmetric, compare from both ends
Signal: problem involves comparing characters from start and end simultaneously

Template:
  int left = 0, right = s.length() - 1;
  while (left < right) {
      // compare s.charAt(left) and s.charAt(right)
      left++;
      right--;
  }
```

---

## 🟢 LC #125 — Valid Palindrome

```java
public boolean isPalindrome(String s) {
    int left = 0, right = s.length() - 1;

    while (left < right) {
        // Skip non-alphanumeric characters
        while (left < right && !Character.isLetterOrDigit(s.charAt(left)))  left++;
        while (left < right && !Character.isLetterOrDigit(s.charAt(right))) right--;

        if (Character.toLowerCase(s.charAt(left)) != Character.toLowerCase(s.charAt(right))) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}
```

## 🟢 LC #344 — Reverse String

```java
public void reverseString(char[] s) {

    while (left < right) {
        char temp = s[left];
        s[left++] = s[right];
        s[right--] = temp;
```

## 🟡 LC #680 — Valid Palindrome II (at most one deletion)

```java
public boolean validPalindrome(String s) {
    int left = 0, right = s.length() - 1;

    while (left < right) {
        if (s.charAt(left) != s.charAt(right)) {
            // Try skipping left char OR right char
            return isPalin(s, left + 1, right) || isPalin(s, left, right - 1);
        }
        left++;
        right--;
    }
    return true;
}

private boolean isPalin(String s, int left, int right) {
    while (left < right) {
        if (s.charAt(left++) != s.charAt(right--)) return false;
    }
    return true;
}
```

## 🟡 LC #151 — Reverse Words in a String

```java
public String reverseWords(String s) {
    // Split, reverse array, join
    String[] words = s.trim().split("\\s+");
    StringBuilder sb = new StringBuilder();
    for (int i = words.length - 1; i >= 0; i--) {
        sb.append(words[i]);
        if (i > 0) sb.append(" ");
    }
    return sb.toString();
}

// In-place char array approach:
public String reverseWordsInPlace(String s) {
    char[] arr = s.trim().replaceAll("\\s+", " ").toCharArray();
    int n = arr.length;
    reverse(arr, 0, n - 1);       // Reverse entire string
    int start = 0;
    for (int i = 0; i <= n; i++) {
        if (i == n || arr[i] == ' ') {
            reverse(arr, start, i - 1); // Reverse each word
            start = i + 1;
        }
    }
    return new String(arr);
}

private void reverse(char[] arr, int l, int r) {
    while (l < r) { char t = arr[l]; arr[l++] = arr[r]; arr[r--] = t; }
}
```

# 6. Pattern 2 — Sliding Window on Strings

## When to Use

```
Keywords: longest/shortest substring, "at most k", "contains all chars of t"
Signal: need a contiguous portion of string satisfying a condition

Template (variable window):
  Map<Character, Integer> window = new HashMap<>();
  int left = 0, maxLen = 0;
  
  for (int right = 0; right < s.length(); right++) {
      // ADD s.charAt(right) to window
      
      while (window violates constraint) {
          // REMOVE s.charAt(left) from window
          left++;
      }
      
      maxLen = Math.max(maxLen, right - left + 1);
  }
```

---

## 🟡 LC #3 — Longest Substring Without Repeating Characters

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> lastSeen = new HashMap<>();
    int maxLen = 0, left = 0;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);

        // If char seen and its last position is inside current window
        if (lastSeen.containsKey(c) && lastSeen.get(c) >= left) {
            left = lastSeen.get(c) + 1;  // Shrink window past the duplicate
        }

        lastSeen.put(c, right);
        maxLen = Math.max(maxLen, right - left + 1);
    }

    return maxLen;
}
```

## 🟡 LC #424 — Longest Repeating Character Replacement

```java
// Find longest substring where you can replace at most k chars to make all same
public int characterReplacement(String s, int k) {
    int[] count = new int[26];
    int left = 0, maxCount = 0, maxLen = 0;

    for (int right = 0; right < s.length(); right++) {
        count[s.charAt(right) - 'A']++;
        maxCount = Math.max(maxCount, count[s.charAt(right) - 'A']);

        // Window size - maxCount = chars that need replacement
        // If > k, window is invalid → shrink
        if (right - left + 1 - maxCount > k) {
            count[s.charAt(left) - 'A']--;
            left++;
        }

        maxLen = Math.max(maxLen, right - left + 1);
    }

    return maxLen;
}
```

## 🔴 LC #76 — Minimum Window Substring

```java
// Find smallest window in s containing all chars of t
public String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>();
    for (char c : t.toCharArray()) need.merge(c, 1, Integer::sum);

    int left = 0, formed = 0, required = need.size();
    int minLen = Integer.MAX_VALUE, minLeft = 0;
    Map<Character, Integer> window = new HashMap<>();

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        window.merge(c, 1, Integer::sum);

        // Check if this char's count now satisfies t's requirement
        if (need.containsKey(c) && window.get(c).equals(need.get(c))) formed++;

        // Shrink window while all chars are satisfied
        while (formed == required) {
            if (right - left + 1 < minLen) {
                minLen = right - left + 1;
                minLeft = left;
            }
            char leftChar = s.charAt(left);
            window.merge(leftChar, -1, Integer::sum);
            if (need.containsKey(leftChar) && window.get(leftChar) < need.get(leftChar)) formed--;
            left++;
        }
    }

    return minLen == Integer.MAX_VALUE ? "" : s.substring(minLeft, minLeft + minLen);
}
```

```java
public boolean isAnagram(String s, String t) {
    if (s.length() != t.length()) return false;
    int[] count = new int[26];
    for (char c : s.toCharArray()) count[c - 'a']++;
    for (char c : t.toCharArray()) {

    }
    return true;
}
```

```java
// Find all start indices of anagram of p in s
public List<Integer> findAnagrams(String s, String p) {
    List<Integer> result = new ArrayList<>();
    if (s.length() < p.length()) return result;

    int[] pFreq = new int[26];
    int[] wFreq = new int[26];
    for (char c : p.toCharArray()) pFreq[c - 'a']++;

    int k = p.length();
    for (int right = 0; right < s.length(); right++) {
        wFreq[s.charAt(right) - 'a']++;

        if (right >= k) {
            wFreq[s.charAt(right - k) - 'a']--;  // Remove leftmost char
        }

        if (Arrays.equals(pFreq, wFreq)) result.add(right - k + 1);
    }
    return result;
}
```

## 🟡 LC #394 — Decode String

```java
// "3[a2[c]]" → "accaccacc"
public String decodeString(String s) {
    Deque<Integer> countStack = new ArrayDeque<>();
    Deque<StringBuilder> strStack = new ArrayDeque<>();
    StringBuilder current = new StringBuilder();
    int k = 0;

    for (char c : s.toCharArray()) {
        if (Character.isDigit(c)) {
            k = k * 10 + (c - '0');  // Handle multi-digit numbers
        } else if (c == '[') {
            countStack.push(k);
            strStack.push(current);
            current = new StringBuilder();
            k = 0;
        } else if (c == ']') {
            int repeat = countStack.pop();
            StringBuilder prev = strStack.pop();
            String inner = current.toString();
            for (int i = 0; i < repeat; i++) prev.append(inner);
            current = prev;
        } else {
            current.append(c);
        }
    }

    return current.toString();
}
```

## 🟡 LC #1047 — Remove All Adjacent Duplicates In String

```java
public String removeDuplicates(String s) {
    Deque<Character> stack = new ArrayDeque<>();

    for (char c : s.toCharArray()) {
        if (!stack.isEmpty() && stack.peek() == c) {
            stack.pop();   // Remove duplicate pair
        } else {
            stack.push(c);
        }
    }

    StringBuilder sb = new StringBuilder();
    while (!stack.isEmpty()) sb.append(stack.pop());
    return sb.reverse().toString();
}

// Cleaner: use StringBuilder as a stack
public String removeDuplicates2(String s) {
    StringBuilder sb = new StringBuilder();
    for (char c : s.toCharArray()) {
        if (sb.length() > 0 && sb.charAt(sb.length() - 1) == c) {
            sb.deleteCharAt(sb.length() - 1);
        } else {
            sb.append(c);
        }
    }
    return sb.toString();
}
```

## 🟡 LC #71 — Simplify Path

```java
public String simplifyPath(String path) {
    Deque<String> stack = new ArrayDeque<>();
    for (String part : path.split("/")) {
        if (part.equals("..")) {
            if (!stack.isEmpty()) stack.pop();
        } else if (!part.isEmpty() && !part.equals(".")) {
            stack.push(part);
        }
    }
    StringBuilder sb = new StringBuilder();
    while (!stack.isEmpty()) sb.insert(0, "/" + stack.pop());
    return sb.length() == 0 ? "/" : sb.toString();
}
```

## 🟡 LC #316 — Remove Duplicate Letters (Greedy + Stack)

```java
// Remove duplicate letters so each appears once, result is smallest lexicographically
public String removeDuplicateLetters(String s) {
    int[] count = new int[26];      // Remaining count of each char
    boolean[] inStack = new boolean[26];
    Deque<Character> stack = new ArrayDeque<>();

    for (char c : s.toCharArray()) count[c - 'a']++;

    for (char c : s.toCharArray()) {
        count[c - 'a']--;

        if (inStack[c - 'a']) continue;  // Already in result

        // Remove larger chars from stack if they appear later
        while (!stack.isEmpty() && stack.peek() > c && count[stack.peek() - 'a'] > 0) {
            inStack[stack.pop() - 'a'] = false;
        }

        stack.push(c);
        inStack[c - 'a'] = true;
    }

    StringBuilder sb = new StringBuilder();
    while (!stack.isEmpty()) sb.append(stack.pop());
    return sb.reverse().toString();
}
```

## 🟢 LC #14 — Longest Common Prefix

```java
public String longestCommonPrefix(String[] strs) {
    if (strs.length == 0) return "";
    String prefix = strs[0];

    for (int i = 1; i < strs.length; i++) {
        // Shrink prefix until strs[i] starts with it
        while (!strs[i].startsWith(prefix)) {
            prefix = prefix.substring(0, prefix.length() - 1);
            if (prefix.isEmpty()) return "";
        }
    }

    return prefix;
}
```

## 🟡 LC #1143 — Longest Common Subsequence (String DP)

```java
// LCS is a string pattern that appears in BOTH strings (not necessarily contiguous)
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i-1) == text2.charAt(j-1)) {
                dp[i][j] = dp[i-1][j-1] + 1;  // Chars match: extend LCS
            } else {
                dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);  // Skip one char
            }
        }
    }

    return dp[m][n];
}
```

## 🟡 LC #5 — Longest Palindromic Substring (Expand Around Center)

```java
// Before Manacher's, use expand-around-center: O(n²)
public String longestPalindrome(String s) {
    int start = 0, maxLen = 1;

    for (int i = 0; i < s.length(); i++) {
        // Odd length palindromes
        int len1 = expand(s, i, i);
        // Even length palindromes
        int len2 = expand(s, i, i + 1);

        int len = Math.max(len1, len2);
        if (len > maxLen) {
            maxLen = len;
            start = i - (len - 1) / 2;
        }
    }

    return s.substring(start, start + maxLen);
}

private int expand(String s, int left, int right) {
    while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
        left--;
        right++;
    }
    return right - left - 1;
}
```

## 🟡 LC #43 — Multiply Strings

```java
public String multiply(String num1, String num2) {
    int m = num1.length(), n = num2.length();
    int[] result = new int[m + n];

    // Multiply digit by digit, accumulate in result array
    for (int i = m - 1; i >= 0; i--) {
        for (int j = n - 1; j >= 0; j--) {
            int mul = (num1.charAt(i) - '0') * (num2.charAt(j) - '0');
            int p1 = i + j, p2 = i + j + 1;
            int sum = mul + result[p2];
            result[p2] = sum % 10;
            result[p1] += sum / 10;
        }
    }

    StringBuilder sb = new StringBuilder();
    for (int digit : result) {
        if (!(sb.length() == 0 && digit == 0)) sb.append(digit);
    }
    return sb.length() == 0 ? "0" : sb.toString();
}
```

# 11. KMP (Knuth-Morris-Pratt) Algorithm

## Why KMP Exists

```
PROBLEM: Find all occurrences of pattern P in text T.

NAIVE approach: slide P over T, check each position → O(n*m)

Example: T = "aaaaab", P = "aaaab"
Naive:
  Position 0: a a a a b  vs  a a a a b → mismatch at last char
              a a a a a
  Position 1:   a a a b  vs  a a a a b → mismatch
                a a a a
  ...wastes all comparisons already made!

KMP idea: When mismatch occurs, DON'T go back to start of pattern.
          Use info about the pattern itself to know where to resume.
          → O(n + m) total!
```

## The LPS Array (Failure Function)

```
LPS = Longest Proper Prefix which is also Suffix

For pattern P = "AAACAAAA":
  P[0] = "A"       → no proper prefix = suffix (length > 0) → LPS[0] = 0
  P[0..1] = "AA"   → "A" is both prefix and suffix → LPS[1] = 1
  P[0..2] = "AAA"  → "AA" is both → LPS[2] = 2
  P[0..3] = "AAAC" → no match → LPS[3] = 0
  P[0..4] = "AAACA"→ "A" matches → LPS[4] = 1
  P[0..5] = "AAACAA" → "AA" matches → LPS[5] = 2
  P[0..6] = "AAACAAA" → "AAA" matches → LPS[6] = 3
  P[0..7] = "AAACAAAA" → "AAA" is longest → LPS[7] = 3

LPS = [0, 1, 2, 0, 1, 2, 3, 3]

When mismatch at P[j], jump to P[LPS[j-1]] instead of P[0].
This skips reprocessing what we already know matches!
```

## KMP Implementation

```java
public class KMP {

    // Build LPS (failure function) array
    // lps[i] = length of longest proper prefix of P[0..i] which is also suffix
    private int[] buildLPS(String pattern) {
        int m = pattern.length();
        int[] lps = new int[m];
        lps[0] = 0;  // Always 0

        int len = 0;  // Length of previous longest prefix-suffix
        int i = 1;

        while (i < m) {
            if (pattern.charAt(i) == pattern.charAt(len)) {
                // Match: extend prefix-suffix
                lps[i] = ++len;
                i++;
            } else {
                if (len != 0) {
                    // Mismatch after some matches: fall back
                    len = lps[len - 1];  // DON'T increment i!
                } else {
                    // No match at all
                    lps[i] = 0;
                    i++;
                }
            }
        }

        return lps;
    }

    // Find all occurrences of pattern in text
    public List<Integer> search(String text, String pattern) {
        List<Integer> result = new ArrayList<>();
        int n = text.length(), m = pattern.length();

        if (m == 0) return result;

        int[] lps = buildLPS(pattern);
        int i = 0;  // Index in text
        int j = 0;  // Index in pattern

        while (i < n) {
            if (text.charAt(i) == pattern.charAt(j)) {
                i++;
                j++;
            }

            if (j == m) {
                // Found pattern at index i - j
                result.add(i - j);
                j = lps[j - 1];  // Continue searching (don't reset j to 0)
            } else if (i < n && text.charAt(i) != pattern.charAt(j)) {
                if (j != 0) {
                    j = lps[j - 1];  // Fall back in pattern
                } else {
                    i++;  // Pattern[0] failed: advance text
                }
            }
        }

        return result;
    }

    public static void main(String[] args) {
        KMP kmp = new KMP();

        // Example 1
        String text = "AAACAAAAACAAACAAAC";
        String pat  = "AAACAAAC";
        System.out.println(kmp.search(text, pat));
        // Output: [0, 9] — pattern found at positions 0 and 9

        // Example 2
        System.out.println(kmp.buildLPS("AAACAAAA"));
        // LPS = [0, 1, 2, 0, 1, 2, 3, 3]
    }
}
```

# 12. Rabin-Karp — Rolling Hash

## Why Rabin-Karp Exists

```
KMP is great for single pattern search.
Rabin-Karp is great when you need to search for MANY patterns at once,
or when you need to find if ANY of a set of strings appears.

KEY IDEA: Instead of comparing characters one by one,
          compute a HASH of the current window and compare hashes.
          → Hash comparison is O(1)
          → Only do actual string comparison when hashes match (to confirm)

ROLLING HASH: When window slides right by one character,
              update hash in O(1) without recomputing from scratch.

Example: Hash of "abc" = 'a'*31² + 'b'*31¹ + 'c'*31⁰
         Hash of "bcd" = 'b'*31² + 'c'*31¹ + 'd'*31⁰

         To get "bcd" from "abc":
         Remove 'a': subtract 'a'*31²
         Shift left: multiply by 31
         Add 'd': add 'd'
         → O(1) update!

TIME: O(n + m) average, O(n*m) worst case (many hash collisions)
ADVANTAGE OVER KMP: Easier to implement multi-pattern search
```

## Rabin-Karp Implementation

```java
public class RabinKarp {

    private static final long MOD  = (long) 1e9 + 7;  // Large prime
    private static final long BASE = 31;               // Small prime

    // Precompute BASE^k % MOD for removing leftmost char
    // Single pattern search
    public List<Integer> search(String text, String pattern) {
        List<Integer> result = new ArrayList<>();
        int n = text.length(), m = pattern.length();
        if (m > n) return result;

        // Compute BASE^(m-1) % MOD
        long power = 1;
        for (int i = 0; i < m - 1; i++) power = power * BASE % MOD;

        // Compute hash of pattern and first window
        long patHash = 0, winHash = 0;
        for (int i = 0; i < m; i++) {
            patHash = (patHash * BASE + (pattern.charAt(i) - 'a' + 1)) % MOD;
            winHash = (winHash * BASE + (text.charAt(i) - 'a' + 1)) % MOD;
        }

        for (int i = 0; i <= n - m; i++) {
            if (patHash == winHash) {
                // Hash matches! Verify actual string (handles hash collisions)
                if (text.substring(i, i + m).equals(pattern)) {
                    result.add(i);
                }
            }

            // Roll the hash: remove leftmost, add next char
            if (i < n - m) {
                winHash = (winHash - (text.charAt(i) - 'a' + 1) * power % MOD + MOD) % MOD;
                winHash = (winHash * BASE + (text.charAt(i + m) - 'a' + 1)) % MOD;
            }
        }

        return result;
    }

    public static void main(String[] args) {
        RabinKarp rk = new RabinKarp();
        System.out.println(rk.search("abcabcabc", "abc")); // [0, 3, 6]
        System.out.println(rk.search("hello world", "world")); // [6]
    }
}
```

```
Pattern: a b c  (m = 3)

Pattern hash: 'a'*961 + 'b'*31 + 'c'*1 = 1*961 + 2*31 + 3 = 1026

  Remove 'a': winHash - 1*961 = 65
  Multiply: 65 * 31 = 2015
  Add 'a' (next char after window): 2015 + 1 = 2016 → "bca" hash

Roll to i=2: "cab"
  Remove 'b': 2016 - 2*961 = 94
  Multiply: 94 * 31 = 2914
  Add 'b': 2914 + 2 = 2916 → "cab" hash → ≠ 1026

Roll to i=3: "abc" → hash = 1026 → FOUND at 3!
```

```java
public String longestDupSubstring(String s) {
    int left = 1, right = s.length() - 1;
    String result = "";

    while (left <= right) {
        int mid = left + (right - left) / 2;
        String dup = hasdup(s, mid);
        if (dup != null) { result = dup; left = mid + 1; }
        else right = mid - 1;
    }
    return result;
}

private String hasdup(String s, int len) {
    long MOD = (long)1e9+7, BASE = 31;
    long hash = 0, power = 1;
    for (int i = 0; i < len; i++) {
        hash = (hash * BASE + s.charAt(i) - 'a') % MOD;
        if (i > 0) power = power * BASE % MOD;
    }
    Map<Long, List<Integer>> seen = new HashMap<>();
    seen.computeIfAbsent(hash, k -> new ArrayList<>()).add(0);
    for (int i = 1; i + len - 1 < s.length(); i++) {
        hash = (hash - (s.charAt(i-1)-'a') * power % MOD + MOD) % MOD;
        hash = (hash * BASE + s.charAt(i+len-1) - 'a') % MOD;
        List<Integer> positions = seen.get(hash);
        if (positions != null) {
            String candidate = s.substring(i, i + len);
            for (int pos : positions)
                if (s.substring(pos, pos + len).equals(candidate)) return candidate;
        }
        seen.computeIfAbsent(hash, k -> new ArrayList<>()).add(i);
    }
    return null;
}
```

# 13. Z-Algorithm

## What Z-Array Means

```
For string S of length n, Z[i] = length of longest substring
starting at S[i] which is also a PREFIX of S.

Example: S = "aabxaa"
         Z = [6, 1, 0, 0, 2, 1]
              ↑                  Z[0] = full string length (convention)
                 ↑               Z[1] = 1  ("a" at pos 1 matches prefix "a")
                    ↑            Z[2] = 0  ("b" doesn't match "a")
                       ↑         Z[3] = 0  ("x" doesn't match "a")
                          ↑      Z[4] = 2  ("aa" matches prefix "aa")
                             ↑   Z[5] = 1  ("a" matches prefix "a")

USE CASE: Pattern P in Text T
  Build: S = P + "$" + T    ($ is a separator not in alphabet)
  Compute Z array of S
  Wherever Z[i] == len(P): pattern starts at T[i - len(P) - 1]

WHY: Z[i] tells us how much of S[i..] matches the beginning of S.
     If Z[i] == len(P), the substring at i IS the pattern.
```

## Z-Algorithm Implementation

```java
public class ZAlgorithm {

    // Compute Z array in O(n)
    public int[] buildZ(String s) {
        int n = s.length();
        int[] z = new int[n];
        z[0] = n;  // Convention

        int left = 0, right = 0;  // Current Z-box: [left, right]

        for (int i = 1; i < n; i++) {
            if (i < right) {
                // i is inside the Z-box
                // z[i-left] is the Z value for the mirrored position
                z[i] = Math.min(right - i, z[i - left]);
            }
            // Extend Z-box from i
            while (i + z[i] < n && s.charAt(z[i]) == s.charAt(i + z[i])) {
                z[i]++;
            }
            // Update Z-box if we extended beyond right
            if (i + z[i] > right) {
                left = i;
                right = i + z[i];
            }
        }

        return z;
    }

    // Pattern search using Z-algorithm
    public List<Integer> search(String text, String pattern) {
        List<Integer> result = new ArrayList<>();
        String combined = pattern + "$" + text;  // $ is separator
        int[] z = buildZ(combined);
        int m = pattern.length();

        for (int i = m + 1; i < combined.length(); i++) {
            if (z[i] == m) {
                result.add(i - m - 1);  // Position in original text
            }
        }

        return result;
    }

    public static void main(String[] args) {
        ZAlgorithm za = new ZAlgorithm();

        // Z array example
        int[] z = za.buildZ("aabxaa");
        System.out.println(Arrays.toString(z)); // [6, 1, 0, 0, 2, 1]

        // Pattern search
        System.out.println(za.search("aabxaaab", "aa")); // [0, 4, 5]
    }
}
```

# 14. Manacher's Algorithm — O(n) Palindrome Search

## Why Manacher's Exists

```
PROBLEM: Find the longest palindromic substring.

NAIVE: Try all O(n²) centers, expand each → O(n²)
EXPAND AROUND CENTER: Same complexity — O(n²)
MANACHER'S: O(n) — the optimal solution!

KEY INSIGHT: Similar to KMP/Z, use previous computations.
When we've found palindromes on the left side, mirror them to avoid recomputation.

TRICK 1: Handle even/odd palindromes uniformly by inserting '#' separators.
  "abba" → "#a#b#b#a#"  (length 2n+1, all palindromes now ODD length)
  "aba"  → "#a#b#a#"

TRICK 2: Maintain a "rightmost palindrome" [center, rightBoundary].
  For each new position, use its mirror to get a lower bound on its palindrome radius.
  Only expand when we go beyond the right boundary.
```

## Manacher's Implementation

```java
public class Manacher {

    public String longestPalindrome(String s) {
        // Transform: "abc" → "#a#b#c#"
        StringBuilder t = new StringBuilder("#");
        for (char c : s.toCharArray()) {
            t.append(c).append('#');
        }
        String T = t.toString();
        int n = T.length();

        int[] p = new int[n];  // p[i] = radius of palindrome centered at i (in T)
        int center = 0, right = 0;  // Current rightmost palindrome's center and right boundary

        for (int i = 0; i < n; i++) {
            int mirror = 2 * center - i;  // Mirror of i around center

            if (i < right) {
                // i is inside the current palindrome
                // Use mirror's palindrome radius as starting point
                p[i] = Math.min(right - i, p[mirror]);
            }

            // Attempt to expand palindrome centered at i
            while (i + p[i] + 1 < n && i - p[i] - 1 >= 0
                   && T.charAt(i + p[i] + 1) == T.charAt(i - p[i] - 1)) {
                p[i]++;
            }

            // Update center and right boundary if we've expanded past right
            if (i + p[i] > right) {
                center = i;
                right = i + p[i];
            }
        }

        // Find the maximum palindrome
        int maxLen = 0, centerIdx = 0;
        for (int i = 0; i < n; i++) {
            if (p[i] > maxLen) {
                maxLen = p[i];
                centerIdx = i;
            }
        }

        // Convert back to original string indices
        int start = (centerIdx - maxLen) / 2;
        return s.substring(start, start + maxLen);
    }

    // Count all palindromic substrings
    public int countSubstrings(String s) {
        // Same setup, but count how many palindromes we find
        StringBuilder t = new StringBuilder("#");
        for (char c : s.toCharArray()) t.append(c).append('#');
        String T = t.toString();
        int n = T.length(), count = 0;
        int[] p = new int[n];
        int center = 0, right = 0;

        for (int i = 0; i < n; i++) {
            int mirror = 2 * center - i;
            if (i < right) p[i] = Math.min(right - i, p[mirror]);
            while (i + p[i] + 1 < n && i - p[i] - 1 >= 0
                   && T.charAt(i + p[i] + 1) == T.charAt(i - p[i] - 1)) {
                p[i]++;
            }
            if (i + p[i] > right) { center = i; right = i + p[i]; }
            // Each center i contributes ceil((p[i]+1)/2) palindromes in original string
            count += (p[i] + 1) / 2;
        }

        return count;
    }

    public static void main(String[] args) {
        Manacher m = new Manacher();
        System.out.println(m.longestPalindrome("babad"));   // "bab" or "aba"
        System.out.println(m.longestPalindrome("cbbd"));    // "bb"
        System.out.println(m.longestPalindrome("racecar")); // "racecar"
        System.out.println(m.countSubstrings("abc"));       // 3
        System.out.println(m.countSubstrings("aaa"));       // 6
    }
}
```

```java
public int myAtoi(String s) {
    int i = 0, n = s.length();
    while (i < n && s.charAt(i) == ' ') i++;  // Skip spaces
    if (i == n) return 0;

    int sign = 1;
    if (s.charAt(i) == '-') { sign = -1; i++; }
    else if (s.charAt(i) == '+') { i++; }

    long result = 0;
    while (i < n && Character.isDigit(s.charAt(i))) {
        result = result * 10 + (s.charAt(i++) - '0');
        if (result * sign > Integer.MAX_VALUE) return Integer.MAX_VALUE;
        if (result * sign < Integer.MIN_VALUE) return Integer.MIN_VALUE;
    }
    return (int)(result * sign);
}
```

```java
public List<String> restoreIpAddresses(String s) {

private void backtrack(String s, int start, List<String> parts, List<String> result) {
    if (parts.size() == 4 && start == s.length()) {
        result.add(String.join(".", parts));
        return;
    }
    if (parts.size() == 4 || start == s.length()) return;

    for (int len = 1; len <= 3; len++) {
        if (start + len > s.length()) break;
        String seg = s.substring(start, start + len);
        if (seg.length() > 1 && seg.charAt(0) == '0') break;  // No leading zeros
        if (Integer.parseInt(seg) > 255) break;
        parts.add(seg);
        backtrack(s, start + len, parts, result);
        parts.remove(parts.size() - 1);
    }
}
```

```java
// Remove k adjacent duplicates repeatedly until no more exist
public String removeDuplicates(String s, int k) {
    // Stack stores [char, count]
    Deque<int[]> stack = new ArrayDeque<>();

    for (char c : s.toCharArray()) {
        if (!stack.isEmpty() && stack.peek()[0] == c) {
            stack.peek()[1]++;
            if (stack.peek()[1] == k) stack.pop();  // Remove k-group
        } else {
            stack.push(new int[]{c, 1});
        }
    }

    StringBuilder sb = new StringBuilder();
    for (int[] item : stack) {
        for (int i = 0; i < item[1]; i++) sb.append((char)item[0]);
    }
    return sb.reverse().toString();
}
```

```java
public int longestValidParentheses(String s) {
    Deque<Integer> stack = new ArrayDeque<>();
    stack.push(-1);  // Base for valid substring length calculation
    int maxLen = 0;

    for (int i = 0; i < s.length(); i++) {
        if (s.charAt(i) == '(') {
            stack.push(i);
        } else {
            stack.pop();
            if (stack.isEmpty()) {
                stack.push(i);  // New base
            } else {

    }
```

## 🔴 LC #126 — Word Ladder II

```java
// BFS for shortest paths + backtrack to reconstruct all paths
public List<List<String>> findLadders(String beginWord, String endWord, List<String> wordList) {
    Set<String> wordSet = new HashSet<>(wordList);
    List<List<String>> result = new ArrayList<>();
    if (!wordSet.contains(endWord)) return result;

    Map<String, List<String>> parents = new HashMap<>();
    Map<String, Integer> dist = new HashMap<>();
    dist.put(beginWord, 0);

    Queue<String> queue = new LinkedList<>();
    queue.offer(beginWord);

    while (!queue.isEmpty()) {
        String curr = queue.poll();
        if (curr.equals(endWord)) break;
        char[] arr = curr.toCharArray();
        for (int i = 0; i < arr.length; i++) {
            char orig = arr[i];
            for (char c = 'a'; c <= 'z'; c++) {
                if (c == orig) continue;
                arr[i] = c;
                String next = new String(arr);
                if (wordSet.contains(next)) {
                    int newDist = dist.get(curr) + 1;
                    if (!dist.containsKey(next)) {
                        dist.put(next, newDist);
                        queue.offer(next);
                        parents.computeIfAbsent(next, k -> new ArrayList<>()).add(curr);
                    } else if (dist.get(next) == newDist) {
                        parents.get(next).add(curr);
                    }
                }
            }
            arr[i] = orig;
        }
    }

    // Backtrack from endWord to beginWord
    if (dist.containsKey(endWord)) {
        backtrack(endWord, beginWord, parents, new LinkedList<>(), result);
    }
    return result;
}

private void backtrack(String word, String begin, Map<String,List<String>> parents,
                        LinkedList<String> path, List<List<String>> result) {
    path.addFirst(word);
    if (word.equals(begin)) {
        result.add(new ArrayList<>(path));
    } else if (parents.containsKey(word)) {
        for (String parent : parents.get(word)) {
            backtrack(parent, begin, parents, path, result);
        }
    }
    path.removeFirst();
}
```

```java
Map<Character, Integer> window = new HashMap<>();

    window.merge(s.charAt(right), 1, Integer::sum);     // Add to window
    while (/* window violates constraint */) {

int expand(String s, int l, int r) {
    while (l>=0 && r<s.length() && s.charAt(l)==s.charAt(r)) { l--; r++; }
    return r - l - 1;  // Length of palindrome
}
// Call for both odd: expand(s,i,i) and even: expand(s,i,i+1)

    if (stack.length() > 0 && stack.charAt(stack.length()-1) == c) {
        stack.deleteCharAt(stack.length()-1);  // Pop
    } else {
        stack.append(c);  // Push
    }
}

int[] buildLPS(String p) {
    int[] lps = new int[p.length()];
    int len = 0, i = 1;
    while (i < p.length()) {
        if (p.charAt(i) == p.charAt(len)) lps[i++] = ++len;
        else if (len != 0) len = lps[len-1];
        else lps[i++] = 0;
```

## ⚡ Critical String Facts for Interviews

```
1. String is IMMUTABLE in Java.
   → NEVER concatenate in a loop. Always use StringBuilder.

2. charAt(i) - 'a' gives 0-25 index for lowercase letters.
   charAt(i) - '0' gives digit value.
   (char)('a' + i) converts back.

3. substring(start, end) is [start, end) — end is EXCLUSIVE.
   Length = end - start.

4. String.split(" ") leaves empty strings for consecutive spaces.
   Use split("\\s+") to split on any whitespace (removes empties).

5. For palindrome problems:
   → Simple check: two pointers
   → Longest palindromic substring: expand around center (easy) or Manacher's (hard interview)
   → Count all palindromes: expand around center, O(n²)

6. For pattern matching:
   → Java: s.contains(pattern) uses built-in (O(n*m) in worst case)
   → Interview: implement KMP for O(n+m)

7. Integer overflow: when computing rolling hash, always use long and MOD.

8. Use int[] freq = new int[26] instead of HashMap when only lowercase letters.
   Much faster: O(1) space, constant-time access.

9. Two strings are anagrams if and only if their sorted forms are equal
   OR their character frequency arrays are equal.

10. For "minimum window" problems, track:
    - How many distinct chars of t are "fully satisfied" (formed)
    - Required = t's distinct chars count
    - Shrink when formed == required
```
