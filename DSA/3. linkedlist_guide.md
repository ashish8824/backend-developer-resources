# 1. What is a Linked List?

A **Linked List** is a linear data structure where elements (called **nodes**) are stored at random memory locations (not contiguous like arrays). Each node holds:
1. **Data** — the value it stores
2. **Pointer** — the address of the next node

```
LINKED LIST:  10 → 20 → 30 → null

┌──────┬────┐    ┌──────┬────┐    ┌──────┬────┐
│  10  │  ●─┼───►│  20  │  ●─┼───►│  30  │null│
└──────┴────┘    └──────┴────┘    └──────┴────┘
 Node 1 (addr 100) Node 2 (addr 450) Node 3 (addr 230)

Each node knows the address of the NEXT node.
Nodes can live ANYWHERE in memory!
```

## The Train Analogy

Think of a linked list as a **train**:
- Each car (node) is connected to the NEXT car only
- To reach Car3, you MUST pass through Car1 and Car2
- Adding/removing a car: just reattach the couplings (pointers)!

# 2. Array vs Linked List

This comparison is asked in **every DSA interview**. Know it cold.

```
Operation            | Array          | Linked List
---------------------|----------------|---------------------------
Access by index      | O(1) direct    | O(n) must traverse
Search by value      | O(n)           | O(n)
Insert at beginning  | O(n) shift     | O(1) just update head
Insert at end        | O(1) amortized | O(n) or O(1) with tail ptr
Insert at middle     | O(n) shift     | O(n) traverse + O(1) insert
Delete at beginning  | O(n) shift     | O(1) just update head
Delete at middle     | O(n) shift     | O(n) traverse + O(1) delete
Memory layout        | Contiguous     | Scattered, dynamic
Cache performance    | Excellent      | Poor (random access)
Size flexibility     | Fixed/costly   | Dynamic, grows easily
```

**USE ARRAY WHEN:** frequent random access, known size, cache performance matters.
**USE LINKED LIST WHEN:** frequent insert/delete at beginning, unpredictable size.

# 3. Types of Linked Lists

## 3.1 Singly Linked List
Each node points to NEXT node only.
```
HEAD
 ↓
[1|→] → [2|→] → [3|→] → [4|null]
```

## 3.2 Doubly Linked List
Each node points to BOTH next and previous.
```
HEAD                                    TAIL
 ↓                                       ↓
[null|1|→] ↔ [←|2|→] ↔ [←|3|→] ↔ [←|4|null]
```

## 3.3 Circular Singly Linked List
Last node's next points back to HEAD.
```
[1|→] → [2|→] → [3|→] → [4|→]
 ↑                              │
 └──────────────────────────────┘
```

## 3.4 Circular Doubly Linked List
Head's prev points to tail; tail's next points to head.

**Java's built-in `LinkedList` is a Doubly Linked List** implementing both `List` and `Deque`.

# 5. Singly Linked List — Complete Implementation

```java
public class SinglyLinkedList {

    static class ListNode {
        int val;
        ListNode next;
        ListNode(int val) { this.val = val; }
    }

    private ListNode head;
    private int size;

    // ═══ INSERT ═══════════════════════════════

    // Add at BEGINNING — O(1)
    public void addFirst(int val) {
        ListNode newNode = new ListNode(val);
        newNode.next = head;   // New node points to old head
        head = newNode;        // Head now points to new node
        size++;
    }

    // Add at END — O(n)
    public void addLast(int val) {
        ListNode newNode = new ListNode(val);
        if (head == null) { head = newNode; size++; return; }
        ListNode curr = head;
        while (curr.next != null) curr = curr.next;  // Walk to last node
        curr.next = newNode;
        size++;
    }

    // Add at INDEX — O(n)
    public void addAt(int index, int val) {
        if (index < 0 || index > size) throw new IndexOutOfBoundsException();
        if (index == 0) { addFirst(val); return; }
        ListNode newNode = new ListNode(val);
        ListNode curr = head;
        for (int i = 0; i < index - 1; i++) curr = curr.next;
        newNode.next = curr.next;  // MUST set this FIRST
        curr.next = newNode;
        size++;
    }

    // ═══ DELETE ═══════════════════════════════

    // Remove from BEGINNING — O(1)
    public int removeFirst() {
        if (head == null) throw new RuntimeException("Empty list");
        int val = head.val;
        head = head.next;
        size--;
        return val;
    }

    // Remove from END — O(n)
    public int removeLast() {
        if (head == null) throw new RuntimeException("Empty list");
        if (head.next == null) { int val = head.val; head = null; size--; return val; }
        ListNode curr = head;
        while (curr.next.next != null) curr = curr.next;  // Stop at second-to-last
        int val = curr.next.val;
        curr.next = null;
        size--;
        return val;
    }

    // Remove by VALUE — O(n)
    public boolean removeByValue(int val) {
        if (head == null) return false;
        if (head.val == val) { head = head.next; size--; return true; }
        ListNode curr = head;
        while (curr.next != null) {
            if (curr.next.val == val) {
                curr.next = curr.next.next;  // Skip the node
                size--;
                return true;
            }
            curr = curr.next;
        }
        return false;
    }

    // ═══ ACCESS ═══════════════════════════════

    public int get(int index) {
        if (index < 0 || index >= size) throw new IndexOutOfBoundsException();
        ListNode curr = head;
        for (int i = 0; i < index; i++) curr = curr.next;
        return curr.val;
    }

    public boolean contains(int val) {
        ListNode curr = head;
        while (curr != null) { if (curr.val == val) return true; curr = curr.next; }
        return false;
    }

    public int size() { return size; }
    public boolean isEmpty() { return size == 0; }

    // ═══ REVERSE — O(n) ═══════════════════════

    public void reverse() {
        ListNode prev = null, curr = head;
        while (curr != null) {
            ListNode next = curr.next;
            curr.next = prev;
            prev = curr;
            curr = next;
        }
        head = prev;
    }

    // ═══ PRINT ════════════════════════════════

    public void print() {
        ListNode curr = head;
        StringBuilder sb = new StringBuilder("HEAD → ");
        while (curr != null) {
            sb.append(curr.val);
            if (curr.next != null) sb.append(" → ");
            curr = curr.next;
        }
        System.out.println(sb + " → null");
    }

    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();
        list.addLast(1); list.addLast(2); list.addLast(3);
        list.addFirst(0);
        list.print();         // HEAD → 0 → 1 → 2 → 3 → null
        list.reverse();
        list.print();         // HEAD → 3 → 2 → 1 → 0 → null
        System.out.println("Size: " + list.size());       // 4
        System.out.println("Contains 2: " + list.contains(2)); // true
    }
}
```

# 6. Doubly Linked List — Complete Implementation

```java
public class DoublyLinkedList {

    static class Node {
        int val;
        Node prev, next;
        Node(int val) { this.val = val; }
    }

    private Node head, tail;
    private int size;

    // ADD AT BEGINNING — O(1)
    public void addFirst(int val) {
        Node newNode = new Node(val);
        if (head == null) { head = tail = newNode; }
        else {
            newNode.next = head;
            head.prev = newNode;
            head = newNode;
        }
        size++;
    }

    // ADD AT END — O(1) with tail pointer
    public void addLast(int val) {
        Node newNode = new Node(val);
        if (tail == null) { head = tail = newNode; }
        else {
            tail.next = newNode;
            newNode.prev = tail;
            tail = newNode;
        }
        size++;
    }

    // REMOVE FROM BEGINNING — O(1)
    public int removeFirst() {
        if (head == null) throw new RuntimeException("Empty list");
        int val = head.val;
        if (head == tail) { head = tail = null; }
        else { head = head.next; head.prev = null; }
        size--;
        return val;
    }

    // REMOVE FROM END — O(1) with tail pointer!
    public int removeLast() {
        if (tail == null) throw new RuntimeException("Empty list");
        int val = tail.val;
        if (head == tail) { head = tail = null; }
        else { tail = tail.prev; tail.next = null; }
        size--;
        return val;
    }

    // REMOVE SPECIFIC NODE — O(1) if you have the node reference!
    public void removeNode(Node node) {
        if (node == head) { removeFirst(); return; }
        if (node == tail) { removeLast(); return; }
        node.prev.next = node.next;
        node.next.prev = node.prev;
        size--;
    }

    public void printForward() {
        Node curr = head;
        System.out.print("null ↔ ");
        while (curr != null) { System.out.print(curr.val + " ↔ "); curr = curr.next; }
        System.out.println("null");
    }

    public static void main(String[] args) {
        DoublyLinkedList list = new DoublyLinkedList();
        list.addLast(1); list.addLast(2); list.addLast(3);
        list.addFirst(0);
        list.printForward();  // null ↔ 0 ↔ 1 ↔ 2 ↔ 3 ↔ null
        list.removeLast();    // O(1)!
        list.removeFirst();   // O(1)!
        list.printForward();  // null ↔ 1 ↔ 2 ↔ null
    }
}
```

**Key advantage of Doubly LL:** `removeLast()` is O(1) because we have a `tail` pointer AND `prev` pointer. In singly LL, removeLast is O(n).

# 7. Circular Linked List — Complete Implementation

```java
public class CircularLinkedList {

    static class Node {
        int val; Node next;
        Node(int val) { this.val = val; }
    }

    private Node tail;  // tail.next = head (the circle!)
    private int size;

    // ADD at END — O(1)
    public void add(int val) {
        Node newNode = new Node(val);
        if (tail == null) { tail = newNode; tail.next = tail; }  // Points to itself
        else {
            newNode.next = tail.next;  // New node → head
            tail.next = newNode;       // Old tail → new node
            tail = newNode;            // Update tail
        }
        size++;
    }

    // REMOVE from BEGINNING — O(1)
    public int removeFirst() {
        if (tail == null) throw new RuntimeException("Empty list");
        Node head = tail.next;
        if (tail == head) { tail = null; }
        else { tail.next = head.next; }
        size--;
        return head.val;
    }

    // PRINT — must stop after size nodes to avoid infinite loop!
    public void print() {
        if (tail == null) { System.out.println("Empty"); return; }
        Node curr = tail.next;  // Start at head
        System.out.print("CIRCULAR: ");
        for (int i = 0; i < size; i++) {
            System.out.print(curr.val + (i < size-1 ? " → " : ""));
            curr = curr.next;
        }
        System.out.println(" → (back to start)");
    }
}
```

# 8. Traversal

```java
// FORWARD TRAVERSAL — O(n)
void traverse(ListNode head) {
    ListNode curr = head;
    while (curr != null) {
        System.out.print(curr.val + " ");
        curr = curr.next;
    }
}

// PRINT IN REVERSE using recursion — O(n) space
void printReverse(ListNode node) {
    if (node == null) return;
    printReverse(node.next);         // Go to end first
    System.out.print(node.val + " "); // Print on the way BACK
}

// CRITICAL TRAVERSAL PATTERNS:
while (curr.next != null) curr = curr.next;          // Stop at LAST node
while (curr.next.next != null) curr = curr.next;     // Stop at SECOND-TO-LAST
for (int i = 0; i < n; i++) curr = curr.next;       // Walk exactly n steps
```

# 9. Insertion — The Critical Pointer Order

```
WRONG ORDER — lose reference to rest of list!
curr.next = newNode;       // [1] → newNode (lost [3]!)
newNode.next = curr.next;  // newNode → newNode (circular!)

CORRECT ORDER — always set new node's next FIRST
newNode.next = curr.next;  // newNode → [3]
curr.next = newNode;       // [1] → newNode → [3] ✅
```

# 10. Deletion

```
CASE 1: Delete HEAD — O(1)
head = head.next;

CASE 2: Delete TAIL — O(n)
Walk to second-to-last: while (curr.next.next != null) curr = curr.next;
curr.next = null;

CASE 3: Delete MIDDLE — O(n) to find, O(1) to delete
Walk to node BEFORE target: curr.next = curr.next.next;

INTERVIEW TRICK (LC #237): Delete node WITHOUT head reference
node.val = node.next.val;   // Copy next value into current
node.next = node.next.next; // Skip next node
```

```java
int length(ListNode head) {

ListNode findMiddle(ListNode head) {
```

# 12. Reversal — The Most Tested Operation

```
STEP-BY-STEP: [1] → [2] → [3] → [4] → null

prev=null, curr=[1]
Step 1: next=[2], [1]→null,  prev=[1], curr=[2]
Step 2: next=[3], [2]→[1],   prev=[2], curr=[3]
Step 3: next=[4], [3]→[2],   prev=[3], curr=[4]
Step 4: next=null, [4]→[3],  prev=[4], curr=null

head = prev = [4]   →   [4]→[3]→[2]→[1]→null ✅
```

```java
// ITERATIVE — O(n) time, O(1) space
ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;  // 1. Save next
        curr.next = prev;           // 2. Reverse pointer
        prev = curr;                // 3. Advance prev
        curr = next;                // 4. Advance curr
    }
    return prev;  // New head
}

// RECURSIVE — O(n) time, O(n) space (call stack)
ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = reverseList(head.next);
    head.next.next = head;  // Make head.next point back
    head.next = null;
    return newHead;
}
```

# 13. Pattern 1 — Fast and Slow Pointers (Floyd's Algorithm)

## What is it?

Two pointers at **different speeds**: slow moves 1 step, fast moves 2 steps.

```
Find middle of [1→2→3→4→5]:
Start:  slow=1, fast=1
Step 1: slow=2, fast=3
Step 2: slow=3, fast=5 (fast.next=null, STOP)
→ slow=3 is the middle ✅

Cycle detection in [1→2→3→4→(back to 2)]:
Start:  slow=1, fast=1
Step 1: slow=2, fast=3
Step 2: slow=3, fast=3  ← THEY MET! Cycle exists! ✅
```

## 🟢 LC #876 — Middle of the Linked List

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;       // 1 step
        fast = fast.next.next;  // 2 steps
    }
    return slow;  // Middle!
}
// Odd length [1,2,3,4,5] → returns 3 (exact middle)
// Even length [1,2,3,4]  → returns 3 (second middle)
```

## 🟢 LC #141 — Linked List Cycle

```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;  // They met → cycle!
    }
    return false;  // Fast reached end → no cycle
}
```

**Why it works:** If there's a cycle, fast enters it first. Since fast gains 1 step per iteration, it will eventually "lap" slow and they'll meet inside the cycle.

## 🟡 LC #142 — Linked List Cycle II (Find Cycle Start)

**The Math:** When slow and fast meet, distance to cycle start from meeting point equals distance from head to cycle start. Start two pointers (one from head, one from meeting point) — they meet at cycle start!

```java
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;

    // Phase 1: Find meeting point
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) break;
    }
    if (fast == null || fast.next == null) return null;  // No cycle

    // Phase 2: Find cycle start
    slow = head;  // Reset one pointer to head
    while (slow != fast) {
        slow = slow.next;  // Both move 1 step
        fast = fast.next;
    }
    return slow;  // Meeting point = cycle start!
}
```

## 🟡 LC #234 — Palindrome Linked List

**Strategy:** Find middle → Reverse second half → Compare

```java
public boolean isPalindrome(ListNode head) {
    // Step 1: Find middle
    ListNode slow = head, fast = head;
    while (fast.next != null && fast.next.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    // slow is at end of first half

    // Step 2: Reverse second half
    ListNode secondHalf = reverse(slow.next);

    // Step 3: Compare
    ListNode p1 = head, p2 = secondHalf;
    boolean result = true;
    while (p2 != null) {
        if (p1.val != p2.val) { result = false; break; }
        p1 = p1.next; p2 = p2.next;
    }

    slow.next = reverse(secondHalf);  // Restore list
    return result;
}

private ListNode reverse(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next; curr.next = prev; prev = curr; curr = next;
    }
    return prev;
}
```

# 14. Pattern 2 — Dummy Head Node

## What is it?

A fake node before the actual head — eliminates special cases when the head itself needs to change.

```
Without dummy:
  if (head == null) ...            ← special case
  if (head.val == target) { head = head.next; }  ← special case

With dummy:
  ListNode dummy = new ListNode(0);
  dummy.next = head;
  // Every node has a predecessor — NO special cases!
  return dummy.next;  // New head
```

## 🟡 LC #203 — Remove Linked List Elements

```java
public ListNode removeElements(ListNode head, int val) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode curr = dummy;

    while (curr.next != null) {
        if (curr.next.val == val) {
            curr.next = curr.next.next;  // Skip — don't advance curr!
        } else {
            curr = curr.next;
        }
    }
    return dummy.next;
}
```

## 🟡 LC #21 — Merge Two Sorted Lists

```java
public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
    ListNode dummy = new ListNode(0);
    ListNode curr = dummy;

    while (list1 != null && list2 != null) {
        if (list1.val <= list2.val) { curr.next = list1; list1 = list1.next; }
        else                         { curr.next = list2; list2 = list2.next; }
        curr = curr.next;
    }
    curr.next = (list1 != null) ? list1 : list2;  // Attach remaining
    return dummy.next;
}
```

# 15. Pattern 3 — In-Place Reversal

## 🟡 LC #92 — Reverse Linked List II (Reverse a subrange)

**Input:** `1→2→3→4→5`, left=2, right=4  →  `1→4→3→2→5`

```java
public ListNode reverseBetween(ListNode head, int left, int right) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode prev = dummy;

    // Walk to node BEFORE left
    for (int i = 1; i < left; i++) prev = prev.next;

    ListNode curr = prev.next;  // First node to reverse

    // "Insert at front" trick — repeat (right - left) times
    for (int i = 0; i < right - left; i++) {
        ListNode next = curr.next;
        curr.next = next.next;   // Detach next
        next.next = prev.next;   // next → current front
        prev.next = next;        // prev → next (new front)
    }
    return dummy.next;
}
```

**Dry run for 1→2→3→4→5, left=2, right=4:**
```
prev=[1], curr=[2]
i=0: next=[3], [2]→[4], [3]→[2], [1]→[3] → 1→3→2→4→5
i=1: next=[4], [2]→[5], [4]→[3], [1]→[4] → 1→4→3→2→5 ✅
```

## 🔴 LC #25 — Reverse Nodes in k-Group

```java
public ListNode reverseKGroup(ListNode head, int k) {
    // Check if k nodes remain
    ListNode node = head;
    int count = 0;
    while (node != null && count < k) { node = node.next; count++; }
    if (count < k) return head;  // Not enough nodes

    // Reverse k nodes
    ListNode prev = null, curr = head;
    for (int i = 0; i < k; i++) {
        ListNode next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    // head is now tail of reversed group
    // Recursively handle next group
    head.next = reverseKGroup(curr, k);
    return prev;  // New head
}
```

# 16. Pattern 4 — Merge Two Lists

## 🔴 LC #23 — Merge K Sorted Lists

```java
import java.util.PriorityQueue;

public ListNode mergeKLists(ListNode[] lists) {
    PriorityQueue<ListNode> pq = new PriorityQueue<>((a, b) -> a.val - b.val);

    // Add head of each list
    for (ListNode list : lists) { if (list != null) pq.offer(list); }

    ListNode dummy = new ListNode(0), curr = dummy;

    while (!pq.isEmpty()) {
        ListNode smallest = pq.poll();
        curr.next = smallest;
        curr = curr.next;
        if (smallest.next != null) pq.offer(smallest.next);
    }
    return dummy.next;
}
// Time: O(n log k), Space: O(k)
```

# 17. Pattern 5 — Two Pass / Length Trick

## 🟡 LC #19 — Remove Nth Node From End of List

**Strategy:** Move fast pointer n+1 steps ahead. When fast reaches end, slow is at the node BEFORE the one to delete.

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode fast = dummy, slow = dummy;

    // Move fast n+1 steps ahead
    for (int i = 0; i <= n; i++) fast = fast.next;

    // Move both until fast reaches end
    while (fast != null) { slow = slow.next; fast = fast.next; }

    // slow is just BEFORE the node to delete
    slow.next = slow.next.next;
    return dummy.next;
}
```

**Dry run for [1→2→3→4→5], n=2:**
```
Move fast 3 steps: fast=[3]
Both move until fast=null:
  fast=[4], slow=[1]
  fast=[5], slow=[2]
  fast=null, slow=[3]  ← just before [4] to delete
slow.next = [5]
Result: 1→2→3→5 ✅
```

```java
public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0), curr = dummy;

        if (l2 != null) { sum += l2.val; l2 = l2.next; }

        curr = curr.next;
    }
    return dummy.next;
```

```java
public ListNode swapPairs(ListNode head) {

    dummy.next = head;
```

## 🟡 LC #148 — Sort List (Merge Sort on LL)

```java
public ListNode sortList(ListNode head) {
    if (head == null || head.next == null) return head;

    // Find middle and split
    ListNode mid = getMid(head);
    ListNode right = mid.next;
    mid.next = null;  // Cut in half

    return merge(sortList(head), sortList(right));
}

private ListNode getMid(ListNode head) {
    ListNode slow = head, fast = head.next;
    while (fast != null && fast.next != null) {
        slow = slow.next; fast = fast.next.next;
    }
    return slow;
}

private ListNode merge(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0), curr = dummy;
    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) { curr.next = l1; l1 = l1.next; }
        else                   { curr.next = l2; l2 = l2.next; }
        curr = curr.next;
    }
    curr.next = (l1 != null) ? l1 : l2;
    return dummy.next;
}
// Time: O(n log n), Space: O(log n) recursion stack
```

## 🟡 LC #138 — Copy List with Random Pointer

```java
public Node copyRandomList(Node head) {
    if (head == null) return null;
    Map<Node, Node> map = new HashMap<>();

    // First pass: create all clone nodes
    Node curr = head;
    while (curr != null) { map.put(curr, new Node(curr.val)); curr = curr.next; }

    // Second pass: set next and random pointers
    curr = head;
    while (curr != null) {
        if (curr.next   != null) map.get(curr).next   = map.get(curr.next);
        if (curr.random != null) map.get(curr).random = map.get(curr.random);
        curr = curr.next;
    }
    return map.get(head);
}
```

## 🟡 LC #61 — Rotate List

```java
public ListNode rotateRight(ListNode head, int k) {
    if (head == null || head.next == null || k == 0) return head;

    // Find length and make circular
    int len = 1;
    ListNode tail = head;
    while (tail.next != null) { tail = tail.next; len++; }
    tail.next = head;  // Make circular

    // Find new tail: len - k%len - 1 steps from head
    k = k % len;
    int stepsToNewTail = len - k;
    ListNode newTail = head;
    for (int i = 1; i < stepsToNewTail; i++) newTail = newTail.next;

    ListNode newHead = newTail.next;
    newTail.next = null;  // Break circle
    return newHead;
}
```

## 🟡 LC #86 — Partition List

```java
public ListNode partition(ListNode head, int x) {
    ListNode lessHead = new ListNode(0), greaterHead = new ListNode(0);
    ListNode less = lessHead, greater = greaterHead;

    while (head != null) {
        if (head.val < x) { less.next = head; less = less.next; }
        else               { greater.next = head; greater = greater.next; }
        head = head.next;
    }
    greater.next = null;          // Terminate greater list!
    less.next = greaterHead.next; // Connect less → greater
    return lessHead.next;
}
```

## 🟡 LC #143 — Reorder List

**Problem:** `L0→L1→…→Ln-1→Ln` becomes `L0→Ln→L1→Ln-1→L2→Ln-2→…`

```java
public void reorderList(ListNode head) {
    if (head == null || head.next == null) return;

    // Step 1: Find middle
    ListNode slow = head, fast = head;
    while (fast.next != null && fast.next.next != null) {
        slow = slow.next; fast = fast.next.next;
    }

    // Step 2: Reverse second half
    ListNode secondHalf = reverseList(slow.next);
    slow.next = null;  // Cut at middle

    // Step 3: Merge alternately
    ListNode first = head, second = secondHalf;
    while (second != null) {
        ListNode tmp1 = first.next, tmp2 = second.next;
        first.next = second;
        second.next = tmp1;
        first = tmp1;
        second = tmp2;
    }
}

private ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next; curr.next = prev; prev = curr; curr = next;
    }
    return prev;
}
```

# 21. Hard Problems

## 🔴 LC #146 — LRU Cache

```java
public class LRUCache {
    static class Node { int key, val; Node prev, next; Node(int k, int v){key=k;val=v;} }

    private int capacity;
    private Map<Integer, Node> map;
    private Node head, tail;  // Dummy head/tail — head=LRU end, tail=MRU end

    public LRUCache(int capacity) {
        this.capacity = capacity;
        map = new HashMap<>();
        head = new Node(0,0); tail = new Node(0,0);
        head.next = tail; tail.prev = head;
    }

    public int get(int key) {
        if (!map.containsKey(key)) return -1;
        Node node = map.get(key);
        moveToTail(node);
        return node.val;
    }

    public void put(int key, int val) {
        if (map.containsKey(key)) {
            Node node = map.get(key); node.val = val; moveToTail(node);
        } else {
            if (map.size() == capacity) {
                Node lru = head.next; remove(lru); map.remove(lru.key);
            }
            Node newNode = new Node(key, val);
            addToTail(newNode); map.put(key, newNode);
        }
    }

    private void remove(Node n) { n.prev.next = n.next; n.next.prev = n.prev; }
    private void addToTail(Node n) {
        n.prev = tail.prev; n.next = tail;
        tail.prev.next = n; tail.prev = n;
    }
    private void moveToTail(Node n) { remove(n); addToTail(n); }
}
```

## 🔴 LC #460 — LFU Cache

```java
public class LFUCache {
    private int capacity, minFreq;
    private Map<Integer,Integer> keyVal, keyFreq;
    private Map<Integer, LinkedHashSet<Integer>> freqKeys;

    public LFUCache(int capacity) {
        this.capacity = capacity;
        keyVal = new HashMap<>(); keyFreq = new HashMap<>();
        freqKeys = new HashMap<>();
    }

    public int get(int key) {
        if (!keyVal.containsKey(key)) return -1;
        increaseFreq(key);
        return keyVal.get(key);
    }

    public void put(int key, int val) {
        if (capacity <= 0) return;
        if (keyVal.containsKey(key)) { keyVal.put(key, val); increaseFreq(key); return; }
        if (keyVal.size() >= capacity) removeMinFreq();
        keyVal.put(key, val); keyFreq.put(key, 1);
        freqKeys.computeIfAbsent(1, k -> new LinkedHashSet<>()).add(key);
        minFreq = 1;
    }

    private void increaseFreq(int key) {
        int freq = keyFreq.get(key);
        keyFreq.put(key, freq+1);
        freqKeys.get(freq).remove(key);
        if (freqKeys.get(freq).isEmpty()) { freqKeys.remove(freq); if (minFreq==freq) minFreq++; }
        freqKeys.computeIfAbsent(freq+1, k -> new LinkedHashSet<>()).add(key);
    }

    private void removeMinFreq() {
        LinkedHashSet<Integer> minKeys = freqKeys.get(minFreq);
        int evict = minKeys.iterator().next();
        minKeys.remove(evict);
        if (minKeys.isEmpty()) freqKeys.remove(minFreq);
        keyVal.remove(evict); keyFreq.remove(evict);
    }
}
```

# 23. How to Identify Linked List Patterns

```
Keywords                     → Pattern
──────────────────────────────────────────────────────
"cycle", "loop"              → Fast & Slow Pointers
"middle node"                → Fast & Slow Pointers
"palindrome"                 → Fast/slow + reverse half
"reverse" (whole)            → Three-pointer iterative
"reverse" (part/k-group)     → Dummy + insert-at-front / recursion
"merge" two sorted           → Dummy + two pointers
"merge k sorted"             → Min-heap (Priority Queue)
"remove nth from end"        → Fast pointer n steps ahead
"intersection"               → Switch lists trick
"remove duplicates"          → Dummy + skip logic
"partition"                  → Two separate lists
"sort linked list"           → Merge sort
"copy with random pointer"   → HashMap old→new clone
"LRU / LFU cache"           → Doubly LL + HashMap
"rotate"                     → Find length + reconnect
"reorder list"               → Mid + reverse + merge

DECISION TREE:
├── Find middle / detect cycle?    → Fast & Slow Pointers
├── Reverse something?             → Three-pointer (prev, curr, next)
├── Head might change / build new list? → Dummy Head Node
├── Two lists?                    → Two pointers / merge
├── K lists?                      → Min-Heap
├── Recursion feels natural?      → Use it (palindrome, reverse)
└── O(1) get/insert/delete?       → Doubly LL + HashMap
```

## ⚡ The 5 Golden Rules

```
1. ALWAYS use a dummy head when the head itself might change.

2. For reversal, save next BEFORE reversing the pointer:
   ListNode next = curr.next;  // SAVE FIRST
   curr.next = prev;           // THEN REVERSE

3. For fast/slow, check (fast != null && fast.next != null)
   to avoid NullPointerException.

4. After any pointer manipulation, verify:
   - No accidental cycles created
   - Last node's .next is null

5. DRAW THE LIST with boxes and arrows on paper.
   Never trace linked list pointer manipulation in your head.
   Every bug in linked list problems comes from not drawing it.
```
